# Portfólio Profissional - Gabriel Doce Panta

Portfólio profissional de alto impacto focado em demonstrar competências técnicas, habilidades estratégicas e diferenciais profissionais nas áreas de manutenção elétrica industrial, automação, eletrotécnica e engenharia elétrica.

## Características

- **Design Profissional**: Visual técnico e moderno com cores industriais (azul, cinza, preto)
- **Totalmente Responsivo**: Adaptado para desktop, tablet e mobile
- **Foco em Resultados**: Demonstra valor prático através de casos reais com métricas mensuráveis
- **Organização Clara**: Estrutura lógica que facilita a navegação e compreensão
- **Tecnologia Moderna**: Desenvolvido com React, TypeScript e Tailwind CSS

## Seções do Portfólio

### 1. Header
Apresentação profissional com informações de contato e disponibilidade

### 2. Proposta de Valor
Responde três perguntas fundamentais:
- Quais problemas eu resolvo
- Como gero valor para a empresa
- Qual meu diferencial profissional

### 3. Habilidades Técnicas
Apresentação visual de competências organizadas em 4 categorias:
- Manutenção e Elétrica Industrial
- Automação e Controle
- Confiabilidade e Dados
- Tecnologia e Programação

### 4. Competências Comportamentais
Destaque para soft skills que transformam competência técnica em resultados

### 5. Habilidades Aplicadas
Casos práticos no formato Problema → Ação → Resultado, com métricas reais de impacto

### 6. Conhecimento Técnico e Formação
Apresentação da base educacional e certificações que sustentam as competências

### 7. Objetivo Profissional
Visão clara de crescimento e áreas de interesse profissional

### 8. Footer
Informações de contato e call-to-action para download do CV

## Tecnologias Utilizadas

- **React 18** - Biblioteca JavaScript para interfaces
- **TypeScript** - Tipagem estática para maior confiabilidade
- **Tailwind CSS** - Framework CSS para estilização rápida e responsiva
- **Lucide React** - Biblioteca de ícones modernos
- **Vite** - Build tool moderna e rápida

## Como Executar

1. Instale as dependências:
```bash
npm install
```

2. Execute em modo desenvolvimento:
```bash
npm run dev
```

3. Gere build de produção:
```bash
npm run build
```

## Personalização

### Atualizar Informações Pessoais

Edite o arquivo `src/components/Header.tsx` para atualizar:
- Nome
- Informações de contato (email, telefone)
- Links de redes sociais

### Modificar Casos Práticos

Edite o arquivo `src/components/PracticalCases.tsx` para adicionar ou modificar casos de sucesso

### Ajustar Habilidades

Edite o arquivo `src/components/TechnicalSkills.tsx` para personalizar competências e níveis de proficiência

### Customizar Cores

As cores principais estão definidas no Tailwind CSS. Para mudanças globais, ajuste:
- `bg-blue-600` - Cor principal
- `bg-slate-900` - Cor escura
- `bg-gray-100` - Cor de fundo clara

## Export para PDF

O portfólio foi otimizado para impressão/export em PDF:
1. Abra o portfólio no navegador
2. Pressione `Ctrl+P` (Windows) ou `Cmd+P` (Mac)
3. Selecione "Salvar como PDF"
4. Ajuste margens e configurações conforme necessário
